Namespace: microsoft.graph

In the request body, supply a JSON representation of the [unsupportedDeviceConfiguration](../resources/unsupporteddeviceconfiguration.md) object.

|deviceManagementApplicabilityRuleDeviceMode|[deviceManagementApplicabilityRuleDeviceMode](../resources/devicemanagementapplicabilityruledevicemode.md)|The device mode applicability rule for this Policy. Inherited from [deviceConfiguration](../resources/deviceconfiguration.md)|

## Response

If successful, this method returns a `201 Created` response code and an [unsupportedDeviceConfiguration](../resources/unsupporteddeviceconfiguration.md) object in the response body.
